# Farcaster Prediction Market Mini App (Production)

## Features
- Live Polymarket data
- Trade execution endpoint
- Ready for Farcaster auth + wallet

## Setup

```bash
npm install
npm run dev
```

## Deployment (Vercel)

1. Push to GitHub
2. Import into Vercel
3. Set environment variables if needed
4. Deploy
